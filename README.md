# Krispy-Track
una plataforma de analítica de reseñas de Google Maps para las 6 tiendas de Krispy Kreme (ParqueSur, Caleido, Princesa, La Gavia, Gran Plaza 2, Plenilunio). Hace scraping de reseñas, las guarda en base de datos y muestra un dashboard con estadísticas (valoración media, sentimiento, ranking de empleados mencionados, ranking de tiendas, etc.).
